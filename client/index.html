TodoApp API Home

Here are instructions for using the todo app API

Requirements

This API provides the following functionality:

  • Public can log in or create an account
  • A user can create todo lists
  • A user can add tasks to their todo lists

The API uses access tokens, which can be placed in the 
Authentication
header.

Routes

There Are a few points to note:

  • The API route is 
    /api
  • The update and delete routes can be used for any route. For example to update a user setting use the syntax of update task but on the 
    /Members/:id
    route.

To test the api in action use the Postman chrome extension. Import this Postman collection to see test all the routes in action. Note, you will have to change the access tokens that are in the file to your own access tokens.
